> Vinyl adapter to filter files that have been modified since given Date or Number

## Install
```
npm i --save vinyl-filter-since
npm test
```


## Usage
> For more use-cases see the [tests](./test.js)

```js
var vinylFilterSince = require('vinyl-filter-since');
```
